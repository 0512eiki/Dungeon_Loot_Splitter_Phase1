    Logic Explanation

    I use one array to store loot entries. Each entry is an object with two properties: name and value. A traditional for loop is used to calculate total loot and another loop is used to render each item into the visible list. Conditional logic with if statements validates party size, loot name, and loot value, and also handles edge cases like trying to split before any loot is added.
